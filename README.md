# 🎓 Student Management System

## 📌 Overview

A Python-based console application developed to efficiently manage student records. The system allows users to perform CRUD (Create, Read, Update, Delete) operations while also providing basic academic analysis such as topper identification and average marks calculation.

---

## 🚀 Features

- ➕ Add new student records
- 📋 View all student records
- 🔍 Search student by ID or Name
- ✏️ Update existing student details
- ❌ Delete student records
- 🏆 Identify the class topper
- 📊 Calculate average marks
- 💾 File-based data storage for persistence

---

## 🛠️ Tech Stack

- Python
- File Handling
- Modular Programming
